# value-invest / buybacks

Static MVP for exploring Korean treasury stock acquisition, disposition, retirement, and holding data.

## Commands

```bash
npm install
npm run build:data
npm run validate:data
npm run test
npm run build
python -m pytest
```

`DART_API_KEY` enables live OpenDART collection. Without it, the build uses fixture data so the GitHub Pages frontend still builds without browser-side API keys.
